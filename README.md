新手入门：<br/>
1.安装成功后，登录的后台地址是：http://域名/admin.php<br/>
2.V3微商城不支持V2的所有扩展模块和插件，我司后期会陆续放出V3版插件。<br/>
3.新增店铺后，点击店铺的“店铺管理”进入管理店铺的页面。需要在店铺里的“配置”“商城基础配置”设置开启注册，前台商城端才能进行用户注册。<br/>
4.php环境推荐php5.5版本或更高版本。<br/>

PHP运行环境新手推荐工具下载地址：<br/>
https://www.apachefriends.org/download.html<br/>
